Navigation Button

CSS selector:

.button-nav

Sample code:

<a href="#" class="button-nav">home</a>

Rendered element:

home

In-text link

CSS selector:

.text-link

Sample code:

<a href="#" class="text-link">read more</a>

Rendered element:

read more
